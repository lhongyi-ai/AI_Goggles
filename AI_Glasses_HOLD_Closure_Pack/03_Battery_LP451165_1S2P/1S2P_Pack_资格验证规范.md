# LP451165 1S2P Pack 资格验证规范（工程初稿）

注意：下列数值是项目建议的初始门槛，必须根据供应商正式规格和 RK3576 峰值波形复核。

## 架构

```text
Cell-L -- branch protection --+
                              +-- Pack Protection / Power Path -- BAT_SYS
Cell-R -- branch protection --+
```

两颗电芯并联前必须同型号、同批次、同老化条件，并由供应商完成配对。不得在实验室把任意不同状态电芯直接并联。

## 建议配对门槛

| 参数 | 建议初始门槛 |
|---|---|
| 开路电压差 | <= 20 mV，越小越好 |
| 实测容量差 | <= 3% |
| DCIR 差 | <= 10% |
| 生产批次 | 同批次 |
| 首次充电状态 | 供应商定义并记录 |
| 温度传感 | 每侧一颗 NTC |

## 支路保护

- 每颗电芯支路预留 PTC、保险丝或 fusible link。
- 额定值必须高于正常单支路峰值，并低于 FPC/线束安全电流。
- 最终额定值在测得 `Cell-L` 和 `Cell-R` 峰值及均流误差后确定。
- Pack 主保护与单支路保护的动作阈值不能相互冲突。

## 测试点

- Cell-L 电压与支路电流。
- Cell-R 电压与支路电流。
- Pack 电压、总电流。
- 两侧 NTC。
- Boost 输入电压和 5 V 输出。

## 资格测试矩阵

| 条件 | 测试 |
|---|---|
| SOC | 100%、50%、10% |
| 温度 | 0°C、25°C、45°C |
| 老化 | 新电芯、100循环后 |
| 负载 | AON持续、SoC启动脉冲、录像、AI+Wi-Fi峰值 |

建议负载波形：

1. AON 基线 10–30 mW，持续5分钟。
2. RK3576启动脉冲，使用实测波形回放。
3. 1080p录像负载30分钟。
4. AI + 编码 + Wi-Fi峰值短脉冲。
5. 回到AON状态，观察电压恢复。

## 记录

- 两颗支路电流差和随温度/SOC变化。
- 最低瞬时 Pack 电压。
- Boost UVLO/复位事件。
- 每颗电芯表面温度。
- 保护器件温升。
- 充电时两侧温差。
- 100次启动成功率。

## HOLD 关闭门槛

- 供应商正式 Pack Specification 已取得。
- 两颗电芯配对规则书面确认。
- 每侧支路保护与 NTC 已定义。
- 0/25/45°C、低SOC和老化样品均不触发异常掉电。
- 均流无持续恶化，单侧温升无异常。
- nPM1300 电池 profile 使用最终 Pack 完成，而不是使用通用模型。
