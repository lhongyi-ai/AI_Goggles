# 外壳设计约束摘要 (AR 运动眼镜) — chip-down 阶段参考

来源：`shell_design_brief.pdf`（工程师机械组，2026-06-26）。
**重要：这份 brief 描述的是最终 chip-down 眼镜，不是当前 CM4 EVT 载板。** 见下"关系说明"。

## ⚠ 关系说明：外壳 brief vs 当前 CM4 载板
- brief 的右镜腿主控是 **自设计 RK3576 chip-down PCB，36×28mm**，塞进 **14×8mm 镜腿截面**。
- 当前 V1 是 **Radxa CM4 模块载板**，仅 CM4 模块本体就 **40×55mm** → **物理上装不进镜腿**。
- 这是**预期的**：CM4 载板 = 台架功能/软件验证（需求 §7.1 不要求塞镜腿）；外壳 brief = chip-down 最终产品（需求 §7.2）。**两块是不同的板。** 当前载板验证功能，chip-down 阶段再按本约束缩小。

## 整机目标
| 项 | 目标 |
|---|---|
| 整机重量 | ≤ 90g（对标 Ray-Ban Meta 49g / 闪极 48g；理想 <50g）|
| 佩戴 | 适配普通眼镜，镜框不过厚，鼻托可调 |
| 防护 | IPX4（运动/防汗）|
| 散热 | 被动，镜腿贴金属支架导热，无风扇，外壳表面 ≤40°C |
| 充电/接口 | 镜腿末端 磁吸 Pogo Pin 或 USB-C |
| 镜腿 | 138–142mm 长，截面 **14×8mm**（或 16×8）|
| 镜框宽 | 143–147mm |

## 结构分区与重量平衡
- **前框**：摄像头、麦克风×2、(显示模块, 待定)、鼻托
- **左镜腿**：电池×2、扬声器  ← 平衡右侧主控重量
- **右镜腿**：主控、电池、扬声器、麦克风
- FPC 过铰链连接左右

## 关键器件尺寸（chip-down 目标）
| 器件 | 尺寸 mm | 厚 | 备注 |
|---|---|---|---|
| 摄像头模组 | 25×25 (参考 IMX415) | 5–8 | 我们的 SC3336(B) 实测 25×24，含镜头 23.95 高 → **EVT 模块偏大，glasses 需薄型 FFC 相机** |
| MEMS 麦克风 ×2 | 3×3 | 2 | 前框两侧，FPC 引出 |
| 主控 PCB | **36×28** | 1.0 | RK3576 **chip-down**；LPDDR5 须在 SoC 15mm 内 |
| 电池 ×2(左)+×1(右) | 35×18 | 4 | 2 颗在左平衡 |
| 扬声器 | 9×20 | 3 | DLS0920A 或贴片 |
| 充电口 | 10×5 | — | Pogo Pin / USB-C |
| 铰链 FPC | 宽 6–10 | 0.1–0.18 | 弯折半径 ≥2mm，寿命 >5万次 |

## 开口/接口位置
充电口=右镜腿末端 10×3；摄像头窗口=前框中央 8×8；麦克风孔=前框/镜腿 Ø1mm×3–4；扬声器出声孔；触摸区=右镜腿外侧。

## 散热
RK3576 满载 ~600mW（峰值 3–10W 瞬态）→ 导热硅胶 → 金属支架 → 镜腿外壳 → 环境；表面 ≤40°C。SoC 区 PCB ~25×20mm 留导热面。

## 对当前 CM4 载板（EVT）有用的 carry-forward
1. **相机 = SC3336(B)**：2-lane / 3.3V / 20-pin（已写入 BOM）
2. 连接器朝向：相机 FPC 朝前框、Wi-Fi/GNSS 天线远离金属铰链、充电/USB-C 在板边
3. 麦克风 2× MEMS 3×3mm（FPC）、扬声器 DLS0920A 9×20mm
4. 电池：~35×18×4mm ×2–3 → 反推容量（支撑 >3–4h 功率预算）
5. 显示模块（Micro-OLED）出现在外壳，但 **V1 载板电子学不含显示**（需求排除 AR 显示）
